## Dir: multivariate_mi_analysis 
  > Note: This directory contains scripts for multivariate mutual information analysis, including data processing, entropy calculation, and visualization.
----------
### Tree 
- [multivariate_mi_analysis/](./)
  - [data_mod.py](./data_mod.py)  - Python script for data modification
  - [entropy_mutualinfo.py](./entropy_mutualinfo.py)  - Python script for entropy and mutual information calculation
  - [mi_parallel_processing.py](./mi_parallel_processing.py)  - Python script for parallel processing of mutual information
  - [mi_red_rmvr.py](./mi_red_rmvr.py)  - Python script for mutual information redundancy removal
  - [multivariate_main_run.py](./multivariate_main_run.py)  - Main script for running multivariate analysis
  - [multivariate_run.py](./multivariate_run.py)  - Script for running multivariate analysis
  - [notebook__init__b.py](./notebook__init__b.py)  - Initialization script for Jupyter notebooks
  - [Readme.md](./Readme.md)  - Readme file for the multivariate_mi_analysis directory
  - [three_mi_value_splitter.py](./three_mi_value_splitter.py)  - Python script for splitting three mutual information values
  - [visualization.py](./visualization.py)  - Python script for visualization
  - [three_var_mi_utils/](./three_var_mi_utils/)
    - [data/](./three_var_mi_utils/data/)
      - [csv_handler.py](./three_var_mi_utils/data/csv_handler.py)  - Python script for handling CSV data
    - [notebooks/](./three_var_mi_utils/notebooks/)
      - [data_processor.py](./three_var_mi_utils/notebooks/data_processor.py)  - Python script for processing data in notebooks
    - [visualization/](./three_var_mi_utils/visualization/)
      - [visulaizer.py](./three_var_mi_utils/visualization/visulaizer.py)  - Python script for visualization

## Running the Main Script

To run the main script for multivariate mutual information analysis, execute the following command:

```sh
python multivariate_main_run.py
```
